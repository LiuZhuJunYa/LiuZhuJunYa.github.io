# 交大胖本地latex安装教学

本来是直接用 overleaf 就不需要本地 latex 了，但是由于 overleaf 会编译超时的问题，所以还是搭建一个本地的会比较友好，教程来源博主交大胖，https://www.bilibili.com/video/BV1p44y1P7P4?spm_id_from=333.788.videopod.sections&vd_source=234a85edef825c67e44c786eaf168908

* texlive 安装：https://www.tug.org/texlive/windows.html#install
* 安装 Sublime Text，要注册码（懂的都懂）

1. 打开 sublime text，点击 Preferences（首选项）-Browse Packages（浏览插件目录），然后回到上一级目录（Sublime Text 3），然后打开Installed Packages文件夹，将03文件夹内文件（Package Control.sublime-package）复制粘贴到此处
2. 重启sublime text，在Preferences下可以看到Package Control，说明上一步骤安装成功
3. 在sublime text界面，Ctrl+Shift+P打开Package Control，输入Package Control: Install Package，出现列表后输入 `LatexTools`，选择它安装，该步骤为sublime安装Latex基础插件
4. 在sublime text界面，Ctrl+Shift+P打开Package Control，输入Package Control: Install Package，出现列表后输入 `Latex-cwl`，选择它安装，该步骤为sublime安装Latex命令自动补全功能

* SumatraPDF 安装：https://www.sumatrapdfreader.org/download-free-pdf-viewer

---

1. (配置latextool). 在sublime text界面，点击Preferences-Package Settings-LatexTools-Settings-User，将05文件中的代码粘贴替换到新打开文件的相应位置，注意不是全文替换！而是只替换 "windows":{相应区域}注意此处的texlive文件夹和sumatra文件夹要改为你实际安装相应软件的位置。继续在文件中搜索并替换类似字段为 "builder": "simple",

   ```
   	"windows": {
   			// Path used when invoking tex & friends; "" is fine for MiKTeX
   			// For TeXlive 2011 (or other years) use
   			// "texpath" : "C:\\texlive\\2011\\bin\\win32;$PATH",
   			"texpath" : "D:\\texlive\\2017\\bin\\win32;$PATH",
   			// TeX distro: "miktex" or "texlive"
   			"distro" : "texlive",
   			// Command to invoke Sumatra. If blank, "SumatraPDF.exe" is used (it has to be on your PATH)
   			"sumatra": "D:\\Program Files\\SumatraPDF\\SumatraPDF.exe",
   			// Command to invoke Sublime Text. Used if the keep_focus toggle is true.
   			// If blank, "subl.exe" or "sublime_text.exe" will be used.
   			"sublime_executable": "",
   			// how long (in seconds) to wait after the jump_to_pdf command completes
   			// before switching focus back to Sublime Text. This may need to be
   			// adjusted depending on your machine and configuration.
   			"keep_focus_delay": 0.5
   		},
   ```

2. (配置SumatraPDF环境变量). 选择此电脑-属性- 高级系统设置-环境变量，在系统变量的Path变量中，加入 SumatraPDF 的安装路径，如 D:\Program Files\SumatraPDF

3. (配置SumatraPDF反向定位功能). Win+R后，输入cmd，执行以下命令（路径根据实际调整） 
   `sumatrapdf.exe -inverse-search "\"D:\Program Files\Sublime Text 3\sublime_text.exe\" \"%f:%l\"`

4. 在SumatraPDF中点 设置-选项，在设置反向搜索命令行中 粘贴以下命令（路径根据实际调整） `"D:\Program Files\Sublime Text 3\sublime_text.exe" "%f:%l"` 配置好后，双击PDF相应位置可以跳转到相应代码

5. (配置sublime).  在sublime text界面，Preference-Settings，在打开界面的右边粘贴替换06文件中的全文，
    该步骤后，就有了单词补全，拼写检查，漂亮的编辑器配色等

6. (设定额外快捷键). 在sublime text界面，Preference-Key Bindings, 在打开界面的右边粘贴替换07文件中的全文，该步骤后，就有了本UP自认为非常必要额外加的快捷键，可以根据文本挨个试试这些快捷键（注意Ctrl+D智能扩选此类快捷键本身sublime就自带，无需配置）

7. (设定snippet). 在sublime text界面，Preference-Browse Packages, 打开Usersnippets文件夹，将08文件夹中的文件全部复制粘贴到此处。各个文件名即为snippet的触发指令，可以挨个试试看。
